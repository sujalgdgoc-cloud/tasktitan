# 🚀 Task Titan – Complete App Guide

---

## 📌 Overview

Task Titan is a **role-based task marketplace app** where:

- **Requesters** post coding tasks with an initial bid  
- **Solvers** compete by placing bids and completing tasks  

### ✨ Features

- 🔄 Real-time bid updates  
- 👤 Role-based dashboards  
- 🌗 Dark/Light mode  
- 📊 Task tracking (Pending / In Progress / Completed)  

---

## 📝 Step 2: Sign Up Process

- Enter:
